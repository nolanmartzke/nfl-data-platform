This project is an end-to-end data engineering pipeline for NFL analytics.  
It ingests real NFL data, transforms it, stores it in a warehouse, and produces
analytics + machine learning predictions.

## Features
- Automated ETL with Apache Airflow
- Containerized environment using Docker
- PostgreSQL data warehouse
- Python ingestion from NFL API
- Data cleaning + feature engineering
- Machine learning models for team ranking and game predictions
- Streamlit dashboard for visualization
- CI/CD pipeline using GitHub Actions

## Tech Stack
- Python (Pandas, Requests, scikit-learn)
- Apache Airflow
- PostgreSQL
- Docker & docker-compose
- Streamlit
- GitHub Actions
